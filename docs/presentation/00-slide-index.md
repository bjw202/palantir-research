# 프레젠테이션 슬라이드 인덱스

> **청중:** 경영진 + 기술팀 (AI/온톨로지 비전문가 포함)
> **작성일:** 2026-03-16

## 발표 흐름

```
Part 1: 왜 바꿔야 하는가 (문제)     → 슬라이드 1~12  [12분]

Part 3: 얼마나, 언제 (전략/투자)    → 슬라이드 25~35 [13분]
```

## 디자인 가이드

### 컬러 팔레트
- **Primary:** 진한 남색 (#1B2A4A) — 표지, 전환 슬라이드
- **Accent:** 파란색 (#2196F3) — 솔루션, After, 긍정
- **Warning:** 빨간색 (#E53935) — 문제, Before, 비용
- **Background:** 밝은 회색 (#F5F5F5) — 일반 슬라이드
- **Text:** 진한 회색 (#333333), 흰색 (#FFFFFF)

### 폰트
- 제목: Pretendard Bold / 28~36pt
- 본문: Pretendard Regular / 18~22pt
- 강조 수치: Pretendard ExtraBold / 48~72pt

### 레이아웃 원칙
- 한 슬라이드 핵심 메시지 1개
- 텍스트 최소화, 숫자와 비주얼로 전달
- Before(빨간/어두운) vs After(파란/밝은) 일관된 색상 코딩
- 발표자 노트에 상세 스크립트 포함
