### C-1. 상황 설정

## 통합 효과 분석

### 3개 시나리오의 시너지 효과

```
┌─────────────────────────────────────────────────────────────────────────┐
│                                                                         │
│   시나리오 A (품질 추적)                                                  │
│   불량 근본 원인 데이터 축적                                              │
│     │                                                                   │
│     │  A→B: 불량 패턴이 이상 감지 모델에 반영                              │
│     │  "Edge Burr 발생 전 진동 패턴" 학습 → 시나리오 B의 이상 감지 정확도 향상 │
│     ▼                                                                   │
│   시나리오 B (이상 대응)                                                  │
│   실시간 이상 데이터 축적                                                 │
│     │                                                                   │
│     │  B→C: 이상 발생 빈도/패턴이 정비 예측 모델에 반영                     │
│     │  "진동 이상 빈도 증가 = 설비 열화 가속" → 시나리오 C의 정비 시점 정확도 향상│
│     ▼                                                                   │
│   시나리오 C (예지정비)                                                   │
│   예방적 정비 실행 데이터 축적                                             │
│     │                                                                   │
│     │  C→A: 정비 전후 품질 변화 데이터가 품질 모델에 반영                    │
│     │  "정비 후 7일간 공차 편차 패턴" → 시나리오 A의 기여도 분석 정확도 향상   │
│     │                                                                   │
│     └──→ 닫힌 루프 완성 (A → B → C → A)                                  │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

#### 데이터 선순환 구조

| 시나리오 | 생성하는 데이터 | 소비하는 시나리오 | 효과 |
|----------|-----------------|------------------|------|
| **A (품질)** | 불량 기여 요인별 가중치, 복합 원인 패턴 | B (이상 감지 임계치 최적화), C (품질 기반 정비 우선순위) | 불량 패턴이 예방 모델 강화 |
| **B (이상)** | 이상 발생~조치~결과 이력, 알람 정확도 피드백 | A (이상 후 불량 발생 상관관계), C (이상 빈도 기반 열화 가속 판단) | 실시간 데이터가 분석 정확도 강화 |
| **C (정비)** | 정비 전후 설비 상태 변화, 정비 효과 지속 기간 | A (정비 직후 품질 편차 패턴), B (정비 후 정상 진동 기준선 재설정) | 정비 효과가 품질/이상 모델 보정 |

#### 공통 온톨로지 기반의 통합

세 시나리오가 동일한 온톨로지 객체(Object)와 관계(Link)를 공유하기 때문에 시너지가 가능하다.

```
■ 공유 객체 예시:

  Machine M-14
  ├── 시나리오 A: calculateDefectContribution()의 입력
  ├── 시나리오 B: calculateMachineHealthScore()의 입력
  └── 시나리오 C: estimateDowntimeImpact()의 입력

  → 하나의 Machine 객체에 대해 세 가지 관점의 Function이 동시에 작동
  → 세 Function의 결과가 서로를 보정 (예: 건강 점수가 낮으면 불량 기여도에 설비 가중치 ↑)

  Lot L-2026-0316-A
  ├── 시나리오 A: Defect → [observedIn] → Lot (불량 추적의 시작점)
  ├── 시나리오 B: Machine → [processedOn] ← Lot (이상 시 영향 범위 파악)
  └── 시나리오 C: WorkOrder → 생산 일정에서 Lot 단위 영향 분석

  → 하나의 Lot가 품질, 이상, 정비 세 관점에서 동시에 추적됨
```

### ROI 프레임워크

#### 1. 품질 비용(Cost of Quality) 절감

| 항목 | 현행 | 목표 | 절감액 (연 매출 1,000억원 기준) |
|------|------|------|------------------------------|
| COPQ 비율 | 매출의 15~20% | 매출의 8~10% | **₩50~100억원/년** |
| 내부 실패 비용 (재작업/폐기) | ₩80억원 | ₩30억원 | ₩50억원 |
| 외부 실패 비용 (반품/클레임) | ₩40억원 | ₩15억원 | ₩25억원 |
| 과잉 검사 비용 | ₩15억원 | ₩5억원 | ₩10억원 |

**근거:** 제조 기업의 COPQ는 평균 매출의 15~20%이며, 세계적 수준 기업은 5% 이하를 달성한다 ([IISE](https://www.iise.org/details.aspx?id=22118), [Parsec](https://www.parsec-corp.com/blog/cost-of-quality)). 숨겨진 품질 비용(Hidden COPQ)은 가시 비용의 3~4배에 달한다 ([Fabrico, 2026](https://www.fabrico.io/blog/cost-of-poor-quality-copq-manufacturing-guide/)).

#### 2. 비계획 다운타임 감소

| 항목 | 현행 | 목표 | 절감액 |
|------|------|------|--------|
| 비계획 다운타임 비율 | 매출의 11% | 매출의 5% 이하 | **₩60억원/년** |
| 연간 비계획 정지 시간 | 설비당 200시간 | 설비당 50시간 | 75% 감소 |
| 시간당 정지 비용 | $22,000 (약 ₩2,900만원) | 동일 | 정지 시간 자체 감소 |

**근거:** Siemens 보고서에 따르면 세계 500대 기업의 비계획 다운타임은 연간 매출의 11%, 총 $1.4조 규모이다 ([Siemens, 2024](https://assets.new.siemens.com/siemens/assets/api/uuid:1b43afb5-2d07-47f7-9eb7-893fe7d0bc59/TCOD-2024_original.pdf)).

#### 3. 의사결정 시간 단축

| 의사결정 유형 | 현행 소요 시간 | 목표 소요 시간 | 비고 |
|---------------|---------------|---------------|------|
| 불량 근본 원인 추적 | 6시간 ~ 3일 | 1분 ~ 15분 | 시나리오 A |
| 공정 이상 대응 | 2시간 45분 ~ 7시간 | 5분 ~ 15분 | 시나리오 B |
| 정비 시점 결정 | 2시간 (수동 협의) | 15분 (AI 제안) | 시나리오 C |
| 정비-생산 일정 조율 | 1~3일 (이메일 + 회의) | 30분 (자동 시뮬레이션) | 시나리오 C |

#### 4. 종합 ROI 추정

```
■ 연간 절감 효과 (매출 1,000억원 규모 제조사 기준):

  품질 비용 절감:        ₩50~100억원
  다운타임 비용 절감:     ₩40~60억원
  정비 비용 최적화:       ₩10~15억원
  인적 자원 효율화:       ₩5~8억원
  납기 페널티 회피:       ₩3~5억원
  ─────────────────────────────
  합계:                 ₩108~188억원/년

■ 투자 비용 (추정):

  플랫폼 구축/라이선스:   ₩15~30억원 (초기)
  연간 운영/유지보수:     ₩5~8억원
  내부 인력 육성:         ₩3~5억원 (6~12개월)
  ─────────────────────────────
  초기 투자:             ₩23~43억원
  연간 운영:             ₩5~8억원

■ ROI 예상:
  - 투자 회수 기간: 3~6개월
  - 3년 누적 ROI: 700~1,200%
```

**산업 벤치마크 참조:**
- 예측 정비 도입 기업 95%가 긍정적 ROI 보고, 27%는 1년 내 투자 회수 ([WorkTrek, 2025](https://worktrek.com/blog/predictive-maintenance-trends/))
- AI 기반 예측 정비 ROI: 12~18개월 내 10:1 ~ 30:1 ([Deloitte](https://www.deloitte.com/us/en/services/consulting/services/predictive-maintenance-and-the-smart-factory.html))
- 글로벌 자동차 제조사: AI 시스템 도입 후 다운타임 45% 감소, $1,500만 절감, OEE 12% 향상 ([Bridgera, 2025](https://bridgera.com/predictive-maintenance-in-manufacturing-how-ai-is-transforming-uptime-costs-safety/))
